## :pushpin: 프로젝트 소개

  ### **개요**

   인스타그램 형태로 만든 간단한 토이 프로젝트

 ### **세부사항**
  - 개발기간 : 2023.6.20~2023.7.24
  - 구현사항 : Spring Security, 로그인/회원가입, 좋아요, 댓글, 구독, 프로필 업로드, 게시글 작성
  - 개발 언어 : JAVA / 개발 툴: Spring Boot / RDBMS : MySQL
